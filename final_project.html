Final Project

This page is for my final project for CSC 106. I created a game titled
"Maze Runner" in which you can select from 3 mazes and solve each. This project
was created using the Khan Javascript and ProcessingJS programming environment.

To play select maze 1, 2, or 3 to solve. After selecting the maze you can
then select a cursor pen color to fill out the maze. Can you solve each maze
and guide my bitmoji out?
